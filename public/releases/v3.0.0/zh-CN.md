# v3.0.0

> 2026-03-08

## Features

### Fragment Mode
- 全新碎片模式，将旅行照片填充到世界地图的国家轮廓中
- 点击地图控件上的地球按钮即可在普通视图和碎片视图之间切换
- 自动 GPS 到国家匹配，将每张照片映射到对应的区域
- 基于 Natural Earth 110m 数据的 GeoJSON 国家边界

### Onboarding Guide — Step 3
- 新增第三步使用引导，高亮碎片模式的地球按钮
- 紫色脉冲光环配合清晰的功能说明
- 中英文双语引导文案

## Bug Fixes

- 碎片模式下定位按钮不再缩放到街道级（zoom 14），改为飞到国家级视角（zoom 3）
- 碎片模式强制最大缩放级别为 5，防止通过滚轮、双击或缩放按钮深入到街道级
