# v1.3.0

> 2026-03-04

## Features

### Replay Configuring Phase
- 新增回放配置阶段 — 播放前可自由浏览地图
- 支持逐段选择交通方式（步行、骑行、驾车、公交、火车、飞机、轮船）
- 播放时头像上显示当前交通方式图标

### Avatar System
- 回放标记点支持选择头像样式：个人头像、预设角色或默认圆点
- 新增头像设置页面，提供多种预设角色（猫、狗、鸟、兔、熊、狐狸、企鹅、熊猫）
- 头像和交通方式图标同步渲染到视频录制画布

### Video Recording Enhancements
- 导出视频中现在包含头像和交通方式图标
- 共享样式常量确保 DOM 和 Canvas 渲染的视觉一致性

## Bug Fixes

- 修复回放配置阶段选择交通方式时地图意外缩放的问题
- 修复播放过程中头像偶尔消失的问题（移除了从零缩放的入场动画）
- 移除播放开始时的强制缩放，保持用户当前地图视图

## Refactor

- 提取共享常量到独立数据模块（`waypointStyle`、`iconPaths`、`presetAvatars`、`transportModes`）
- 新增 `haversine` 距离工具函数用于计算路段距离
- 新增 `replay.ts` 类型定义（路段元数据、交通方式）
