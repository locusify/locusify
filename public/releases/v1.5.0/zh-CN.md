# v1.5.0

> 2026-03-06

## Features

### 地图快捷菜单 — 右键 & 长按
- 在地图任意位置右键（桌面端）或长按（移动端）即可添加照片到该坐标
- 照片将放置在所选位置，无需照片本身包含 EXIF GPS 数据
- 仍会从 EXIF 元数据中提取相机信息和拍摄日期
- 右键菜单采用与其他地图浮动面板一致的毛玻璃设计风格
- 回放模式下自动禁用右键菜单

## Bug Fixes

- 修复因 MapLibre 的 `preventDefault()` 与 Radix ContextMenu 事件处理冲突导致右键菜单无法在地图画布上弹出的问题
- 修复菜单项悬停颜色错误使用橙色强调色，改为项目统一的中性填充色
- 修复菜单背景使用未定义的 shadcn 主题变量，改为项目统一的毛玻璃面板设计
