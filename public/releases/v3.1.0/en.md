# v3.1.0

> 2026-03-12

## Features

### Polaroid Photo Panel
- Redesigned replay photo panel with polaroid-style ticket cards, tape decorations, and photo filters
- Frosted glass background with paper texture for an immersive journal feel
- Journal-style captions below the polaroid with serif font and smooth transition animations
- Random motivational quotes that rotate with each photo transition
- Light/dark mode support for all text elements

### Language Switch
- Added language switch to the login drawer for easier locale toggling
- Video and Live Photo support in media uploads

## Bug Fixes

- Fixed context menu overflow and Live Photo pairing via right-click
- Hidden OAuth button text on mobile, showing only icons for cleaner layout

## Refactor

- Unified LazyMedia props across map and replay components
- Replaced ReplayPhotoCard and ReplayTextOverlay with modular PhotoPanel architecture
