# v3.1.0

> 2026-03-12

## Features

### 拍立得照片面板
- 全新设计的轨迹回放照片面板，采用拍立得风格卡片，配有胶带装饰和滤镜效果
- 毛玻璃背景搭配纸张纹理，营造沉浸式日记质感
- 拍立得下方展示衬线字体的日记风格文案，配有平滑过渡动画
- 每次照片切换时随机展示旅行心情文案
- 全部文字元素支持明暗模式适配

### 语言切换
- 登录抽屉新增语言切换入口，方便快速切换语言
- 支持视频和 Live Photo 媒体上传

## Bug Fixes

- 修复右键菜单溢出及 Live Photo 右键配对问题
- 移动端隐藏 OAuth 按钮文字，仅显示图标以优化布局

## Refactor

- 统一地图和回放组件中 LazyMedia 的 props 接口
- 以模块化的 PhotoPanel 架构替代 ReplayPhotoCard 和 ReplayTextOverlay
